# Lesson 11: Final Project — Building / Teacher Script
# 第十一课：毕业项目 — 构建 / 教师逐字稿

**Duration / 时长**: 40 minutes / 分钟
**Materials / 材料**: Slides (`slides.html`), Interactive webpage (`index.html`), Student laptops
**Goal / 目标**: Students spend the full session building their final projects with all AI tools available, with a mid-session check-in to ensure they are on track.
**目标**: 学生利用所有可用的 AI 工具花整节课构建毕业项目，中途进行进度检查以确保按计划推进。

---

## Pre-Class Preparation / 课前准备

> **Important / 重要:**
> - Review each student's saved plan from Lesson 10 (check localStorage or ask them)
> - Make sure the Lesson 11 interactive page is working and all tool tabs function correctly
> - Set a 20-minute timer for the mid-session check-in
> - Be ready to provide hands-on support — this is the most guidance-intensive session
> - Have a backup plan if a student's project idea isn't working (help them pivot to something achievable)
>
> 课前准备：
> - 回顾每个学生在第10课保存的计划（检查 localStorage 或询问他们）
> - 确保第11课互动页面正常运行，所有工具标签功能正常
> - 为中途检查设置20分钟计时器
> - 准备好提供实际指导——这是最需要指导的一节课
> - 如果学生的项目想法不可行，准备备用方案（帮助他们转向可实现的目标）

## Quick Launch / 快速启动 (5 minutes / 分钟)

### [SLIDE 1 — Title] *(0:00)*

EN: "Welcome to BUILD DAY! This is one of the most exciting classes — today you take all your skills and put them to work on YOUR project. Let's make something amazing."

CN: "欢迎来到构建日！这是最令人兴奋的课程之一——今天你将把所有技能运用到你自己的项目上。让我们创造出令人惊叹的作品。"

---

### [SLIDE 2 — Quick Review] *(0:01)*

EN: "Before you start building, I want you to take two minutes to re-read your plan from last class. It should appear at the top of the Lesson 11 page. Read through it — does it still make sense? Do you want to change anything? Maybe you had a better idea since then."

CN: "在你开始构建之前，我想让你花两分钟重新阅读上节课的计划。它应该显示在第11课页面的顶部。通读一遍——还有意义吗？你想改什么吗？也许从那以后你有了更好的想法。"

EN: "If your plan didn't save, that's okay — just tell me what you're building and we'll figure it out."

CN: "如果你的计划没有保存，没关系——告诉我你要构建什么，我们会想办法的。"

---

### [SLIDE 3 — Building Tips] *(0:02)*

EN: "Three tips for building day. First: iterate. That means build a rough version first — don't worry about making it perfect. Just get the basic idea working, then improve it step by step."

CN: "构建日的三个建议。第一：迭代。意思是先构建一个粗略版本——不用担心让它完美。先让基本想法运行起来，然后一步一步改进。"

EN: "Second: save often. Use the project notes area to write down what you've done and what you still need to do. Export your work if you can."

CN: "第二：经常保存。使用项目笔记区域记录你做了什么以及还需要做什么。如果可以的话导出你的工作。"

EN: "Third: experiment! Don't be afraid to try new things. If something doesn't work, you can always undo it or try a different approach."

CN: "第三：实验！不要害怕尝试新东西。如果某些东西不起作用，你随时可以撤销或尝试不同的方法。"

---

### [SLIDES 4-5 — Skip for now] *(0:03)*

EN: "I'll show you the checkpoint slide later. For now, let's get building! Open your laptops, go to the Lesson 11 page, and start working. You have all four tools in tabs at the top: Text AI, Image AI, Code AI, and My Gallery. Switch between them as needed."

CN: "检查点幻灯片我稍后再展示。现在，让我们开始构建！打开电脑，进入第11课页面，开始工作。顶部的标签里有所有四个工具：文字 AI、图像 AI、代码 AI 和我的画廊。根据需要在它们之间切换。"

## Building Phase 1 / 构建第一阶段 (17 minutes / 分钟) *(0:03–0:20)*

**[Switch from slides to monitoring student screens / 从幻灯片切换到巡视学生屏幕]**

EN: "Alright, you have about 17 minutes before our mid-session check-in. Focus on the core of your project first — the most important piece. Don't worry about polish yet."

CN: "好，你有大约17分钟时间到我们的中途检查。先专注于你项目的核心——最重要的部分。先不用担心润色。"

> **Active guidance strategy / 主动指导策略:**
> EN: During this phase, your main job is to walk between students and:
> 1. Check they understand how to use the tab interface
> 2. Help them translate their plan into concrete actions
> 3. Unblock them if they're stuck on a tool
> 4. Help scope down if they're trying to do too much
>
> CN: 在此阶段，你的主要工作是在学生之间走动并：
> 1. 检查他们是否理解如何使用标签界面
> 2. 帮助他们把计划转化为具体行动
> 3. 如果他们在某个工具上卡住了就帮助解决
> 4. 如果他们试图做太多就帮助缩小范围

> **Contingency / 应对方案:**
> EN: If a student is overwhelmed and doesn't know where to start: "Let's look at your plan together. What's step 1? Let's just focus on that one step right now. Which tool do you need for it?"
> CN: 如果学生不知所措不知从哪开始："让我们一起看看你的计划。第一步是什么？让我们现在只专注于那一步。你需要哪个工具？"

> **Contingency / 应对方案:**
> EN: If a student's project idea isn't working out: "That's totally okay — sometimes plans change. Let's think about what you CAN do with the time we have. What's the simplest version of your idea that would still be cool?"
> CN: 如果学生的项目想法不可行："完全没关系——有时候计划会变化。让我们想想在我们有的时间里你能做什么。你的想法的最简单版本是什么样的，仍然很酷？"

> **Contingency / 应对方案:**
> EN: If a student finishes their core project quickly: "Awesome! Now think about what would make it even better. Can you add more images? More detail? A surprise element? Polish the writing?"
> CN: 如果学生很快完成了核心项目："太棒了！现在想想什么能让它更好。你能添加更多图片吗？更多细节？一个惊喜元素？润色文字？"

## Mid-Session Check-in / 中途检查 (3 minutes / 分钟) *(0:20–0:23)*

### [SLIDE 4 — Mid-Session Checkpoint]

**[Bring attention back to the front / 让注意力回到前面]**

EN: "Okay everyone, let's pause for a quick check-in. We're halfway through building time. I have four questions for each of you."

CN: "好了大家，让我们暂停一下做个快速检查。我们构建时间已经过半了。我有四个问题要问你们每个人。"

EN: "First — what have you built so far? Give me a quick one-sentence update."

CN: "第一——你到目前为止构建了什么？给我一个快速的一句话更新。"

**[Student 1 shares / 学生1分享]**

**[Student 2 shares / 学生2分享]**

EN: "Second — what still needs to be done?"

CN: "第二——还需要做什么？"

**[Both students share / 两个学生分享]**

EN: "Third — do you need help with anything? Is there a tool that's not working or something you're stuck on?"

CN: "第三——你需要什么帮助吗？有什么工具不好用或者你卡住的地方吗？"

**[Address any issues / 解决任何问题]**

EN: "And fourth — is your scope still realistic? Remember, next class is polish and present. You don't have another full build session. So finish the core today."

CN: "第四——你的范围还现实吗？记住，下节课是润色和演示。你没有另一整节课的构建时间了。所以今天完成核心部分。"

---

## Building Phase 2 / 构建第二阶段 (15 minutes / 分钟) *(0:23–0:38)*

### [SLIDE 5 — Keep Going!]

EN: "You've got this! Use the rest of the time to keep building. Remember to switch between tabs if you need different tools. And write notes to track your progress!"

CN: "你可以的！利用剩下的时间继续构建。如果需要不同的工具记得在标签之间切换。还有写笔记来追踪你的进度！"

**[Continue walking around and helping students / 继续走动并帮助学生]**

> **At 0:35, give a 3-minute warning / 在 0:35 给出 3 分钟提醒:**
> EN: "Three minutes left! Start wrapping up what you're working on. Make sure to save your project notes!"
> CN: "还有三分钟！开始收尾你正在做的事情。确保保存你的项目笔记！"

---

## Wrap-up / 课程总结 (2 minutes / 分钟) *(0:38–0:40)*

### [SLIDE 6 — Save Your Work]

EN: "Time's up for today! Before you close anything, let's make sure everything is saved."

CN: "今天的时间到了！在你关闭任何东西之前，让我们确保一切都保存了。"

EN: "Check these boxes: Did you save your project notes? Did you export anything important? Did you write down what you still need to do? Good."

CN: "检查这些项目：你保存了项目笔记吗？你导出了重要的东西吗？你写下了还需要做的事情吗？好的。"

EN: "Next class is our LAST class — it's polish and presentation day! You'll have about 20 minutes to add final touches, and then you'll present your project. Start thinking about what you want to say: what is your project, how did you make it, and what did you learn."

CN: "下节课是我们的最后一堂课——润色和演示日！你将有大约20分钟来添加最后的修饰，然后你将展示你的项目。开始想想你想说什么：你的项目是什么，你怎么做的，你学到了什么。"

EN: "Great work today! You should all be really proud of what you're building."

CN: "今天干得好！你们都应该为自己正在构建的东西感到自豪。"
