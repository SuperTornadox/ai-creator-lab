Two Ways to Animate 🛠️

🎨 CSS Animations

- Move HTML elements around
- Great for: buttons, text, simple shapes
- Uses: transform, transition, @keyframes
- Easier for simple effects

🖼️ Canvas / JavaScript

- Draw pixels directly on screen
- Great for: particles, starfields, games
- Uses: canvas, requestAnimationFrame
- More powerful for complex animations

Don't worry — AI handles the code. You describe the vision!

Animation Vocabulary 💬

Use these MOTION WORDS in your descriptions:

🏀 bounce
🌀 spin
🌫️ fade
💜 pulse
🌍 orbit
🧬 morph
Pro tip: "A ball that bounces and pulses with color" is way better than just "a moving ball"

Same Description → Different Results! 🎲

Give AI the same prompt twice and you get different animations:

🎲 Result A

Blue particles with smooth trails, calm floating motion, circular patterns

🎲 Result B

Rainbow particles with sharp edges, energetic bouncing, scattered patterns

Both came from: "colorful particles floating around the screen"
Every generation is unique — that's part of the fun!
