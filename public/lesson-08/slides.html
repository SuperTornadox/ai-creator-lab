How to Describe a Game to AI

  • ✅ Be specific — "3 lives, not 10"
  • ✅ Describe controls — "move with arrow keys"
  • ✅ Explain scoring — "get +10 for each catch"
  • ✅ Set the mood — "bright colors, fun for kids"
  • ❌ Avoid vague — "make something cool" 🚫

The more details you give, the better the game!
